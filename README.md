## Installation

`libxkbcommon-dev` is required by `smithay-client-toolkit`

```sh
sudo apt install libxkbcommon-dev just
git clone https://github.com/rwxroot/cosmic-ext-applet-sysinfo.git
cd cosmic-ext-applet-sysinfo
just build
sudo just install
```
